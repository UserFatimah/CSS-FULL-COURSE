Advanced selectors

CSS Advanced selectors are a set of powerful and flexible selectors that allow you to target and style HTML elements more precisely based on their attributes, relationships, or states

types of advanced selectors

adjacent sibling selectors
the adjacent sibling selector will only select sibling elements directly following after another sibling element, instead of using the title character. with the use of '+' character to select the co-current element
child selector
the child selector only targets the direct children of a given element. the child selector is represented by the sign ' > '.
the descendant selector
the descendant selector creates a style rule where the elements are choosen based on where they are in the hierachy of the page.
the universal selector
universal selector is used to select all the element on the HTML page. universal selector is indicated by an asterisk (*).
attribute selector
attribute selctor is used to select all the element that have specified attribute and applies the CSS property to that attribute.
