css selectors

a CSS selector is the first part of a CSS rule. it is a pattern of element and other terms that tell the browser which HTML element should be selected or have the CSS property values inside the rule applied to them.

basic types of css selectors

the element selector
the element selectors is defined as a selector that selects all the HTML elements by specified name on the webpage.
the class selectors
a class selector is formatted as a period (.) character followed by the name of the class. it selects all element with that class attribute so that unique css declarations can be applied to those specific elements without affecting other element on the page.
the ID selector
the id of an element is unique within a page, so the id selector is used to select one unique element! to select an element with a specific id, write a hash (#) character. followed by the id of the element
