FLOAT AND DISPLAY TYPE PROPERTY

what is float in CSS?
the floats is used for positioning and formatting content.it places an element on the left or right side of its container, allowing text and inline elements to wrap around it. the element is removed from the normal flow of the container. though still remaining a part of the flow in contrast to absolute positioning
what is display property in CSS?
the display property specifies if/how an element is displayed because every HTML element has a default display value depending on what type of element it is

types of display property

property value
value description
inline the display option of inline enables the element to fit itself on the same line.
block an element that has the display property set to block starts on a new line and takes up the available screen width. it is possible to specify the width and height of such elements
inline-block inline-block is used to display an element as an inline-level block container. the element itself is formatted as an inline element, but it can apply height and width values. it is placed as an inline element (on the same line as adjacent content)
inline-flex the inline-flex does not make flex items display inline, it makes the flex container display inline.
