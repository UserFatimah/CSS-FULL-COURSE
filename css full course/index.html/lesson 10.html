FLEXBOX

Flexbox is a layout model in CSS that allows you to design a flexible and efficient way to distribute space and align items with a container. Flexbox is a one-dimensional layout method for arranging items in rows and columns.

Fact(s) about Flexbox

  • Flex box stands for flexible box
  • It is a display type that comes with a range of properties allowing you to arrange items easily
  • It is an alternative to using displays, and other layout properties
  • A flexbox element is split into two main parts: the container, and the items
    1. the container is the parent element in which the display type is active, this is usually in the form of a div.
    2. flex items are child element of the container, and make up the contents of the box.

Flexbox Properties

NOTE: display: flex; must be activated in the flex container because it establishes a new flex formatting context for its children.

Flex direction
The flex-direction property defines the direction in which the flex container's main axis runs. It determines how flex items are placed in the flex container.
Flex wrap
The flex-wrap is used to control the wrapping behaviour of flex items within the flex container. It defines whether flex items are forced onto one line or can wrap onto multiple lines.
Justify content
The justify-content defines how the flex items are aligned along the main axis. It helps distribute extra space in the container, or handle situations where there is not enough space.
Align-items
The align-items defines how the flex items are aligned along the cross axis. It determines the default behavior for how items are positioned in the container perpendicular to the main axis
