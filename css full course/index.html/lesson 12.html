TRANSFORM PROPERTY

The transform property is used to apply transformations to elements. The transform property allows you to modify the apperance and position of an element without affecting the layout of the document

Types of Transform Property

  • translate
  • rotate
  • scale
  • skew
translate property
the translate property allows you to transfer an element from one place to another along the x (horizontal) axis, the Y (vertical) axis, and the Z (depth) axis similar to how your might think of moving an element using offset, like top, bottom, right, and left
rotate property
the rotate property allows you to rotate element. the rotate property defines a value for how much an elements is rotated clockwise around z-axis
the scale property
the scale property defines values for how much an element is scaled in x-axis and y-directions. you can also define how much an element is scaled in z-direction.
the skew
this transformation is a shear mapping (transvection) that distorts each point within an element by a certain angle in the horizontal and vertical directions. the effect is as if you grabbed each corner of the element and pulled them along a certain angle.
